# EV Charging Station Simulation

This package contains the core simulation logic for modeling EV charging station behavior over time.

## Installation

```bash
pip3 install -e .
```

This installs development dependencies including pytest for testing.

## Usage

To run the simulation, use the following command:

```bash
python main.py
```

To run the simulation of concurrency factor, use the following command:

```bash
python main.py --concurrency-factor
```
