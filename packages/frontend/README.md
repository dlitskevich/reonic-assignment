# Frontend Package

React + TypeScript + Tailwind CSS frontend application.

## Development

Run the development server:

```bash
pnpm dev
```

Or from the root:

```bash
pnpm dev:frontend
```

## Build

Build for production:

```bash
pnpm build
```

Or from the root:

```bash
pnpm build:frontend
```

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Build tool and dev server
